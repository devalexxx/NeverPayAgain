# NeverPayAgain
## Setup
To setup the project, you must use a Unity 2022.3 LTS.
1. Clone the repository.
2. Install package dependancies. Download them from unity asset store and import them in `Import` folder.
    * [SharpUI GUI](https://assetstore.unity.com/packages/2d/gui/sharpui-gui-195933)
    * [UniRx](https://assetstore.unity.com/packages/tools/integration/unirx-reactive-extensions-for-unity-17276)
3. Run the main scene
